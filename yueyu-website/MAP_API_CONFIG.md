# 地图API配置说明

## 高德地图API密钥获取步骤

1. **注册高德开发者账号**
   - 访问 [高德开放平台](https://lbs.amap.com/)
   - 点击右上角"注册"按钮
   - 使用手机号或邮箱注册账号

2. **创建应用**
   - 登录后进入控制台
   - 点击"应用管理" → "我的应用"
   - 点击"创建应用"
   - 填写应用信息：
     - 应用名称：青岛悦途国际劳务合作网站
     - 应用类型：选择"其他"或"Web服务"
     - 选择权限：勾选"Web服务API"下的"地图服务"

3. **获取API密钥**
   - 应用创建成功后，在应用列表中找到刚创建的应用
   - 点击"查看"或"设置"
   - 复制"Key"值（就是API密钥）

4. **配置前端代码**
   - 在 `/frontend/src/views/About.vue` 文件中找到第193行：
   - 将 `YOUR_AMAP_KEY` 替换为你获取的实际API密钥
   - 例如：`script.src = 'https://webapi.amap.com/maps?v=2.0&key=abcdef123456789&plugin=AMap.Scale,AMap.ToolBar'`

## 免费额度说明

高德地图为开发者提供了充足的免费额度：

- **Web服务API**：每月 100,000 次免费调用
- **JavaScript API**：每天 50,000 次免费调用
- **地图显示**：无限制免费使用
- **基础功能**：地图显示、缩放、标记、信息窗体等

对于公司官网展示用途，这些免费额度完全够用。

## 注意事项

1. **安全保护**：请勿将API密钥直接提交到公共代码仓库
2. **域名白名单**：在高德控制台设置允许的域名，防止盗用
3. **监控用量**：定期在控制台查看API调用情况，避免超出免费额度
4. **生产环境**：部署时建议使用HTTPS协议
